# はじめてぶろぐ

超初心者向けに作った、すぐ動くブログサイトです。

## できること

- サイトを開いたとき、未ログインならログインページへ移動
- 新規登録ページとログインページを分離
- ユーザー名変更ページとパスワード変更ページを追加
- 名前とパスワードで新規登録
- ユーザーごとに丸いアイコン画像を設定
- 登録したアカウントでログイン
- ログインした人の名前で記事を投稿
- 自分の投稿だけ削除できる
- 記事一覧で投稿者とアイコンも表示
- 全員の投稿、自分の投稿、フレンドと自分の投稿で絞り込み

## 使い方

1. `index.html` をダブルクリックします。
2. 初回は自動で `ログインページ` が開くので、必要なら `新規登録ページ` に進みます。
3. 表示名とパスワードを作ります。
4. 必要なら登録時に画像を選ぶと、丸いアイコンになります。
5. ログインすると自動でホームに戻ります。
6. 記事を書いて `投稿する` を押すと記事一覧に追加されます。

## 大事な注意

- サーバーは使っていません。
- アカウント情報、投稿、フレンド情報は、このブラウザの保存機能に入ります。
- つまり、このパソコン、このブラウザの中だけで使う練習用サイトです。

## ファイル説明

- `index.html` : ブログ本体
- `register.html` : 新規登録
- `login.html` : ログイン
- `change-name.html` : ユーザー名変更
- `change-password.html` : パスワード変更
- `styles.css` : デザイン
- `script.js` : 登録、ログイン、アイコン変更、名前変更、パスワード変更、投稿の動き

## 補足

- ローカル版なので、データはこのブラウザの中だけに保存されます。
- あとから本番版に戻したくなったら、別の保存先に切り替える形でまた育てられます。
